OneCheck

All tests must be saved with the Work Order number as the file name

  • Run OneCheck from the ONX meter at each completed job
    1. At the Ground Block, and CPE
    2. Make certain the correct location limit plan is selected for the location of the test
    3. If you receive any failures or errors - troubleshoot, resolve the issue and run another OneCheck to confirm
  • If the OneCheck test fails due to a suspected maintenance issue: take reading from tap, and discuss the issue with your supervisor before opening a line referral ticket. A OneCheck capture from the Tap must be completed for any line referrals.
  • If the customer has internet services:
    1. Run a speed test out of Ethernet port of modem/router
    2. If speed is not what the customer is paying for – troubleshoot to resolution
  • At the completion of the job – sync your meter with Stratasync. DO NOT wait until the end of the week or end of the day

Any test/check other than the OneCheck requires the tech to hit the “STOP” button to properly save the data
